# english-plus

## 1. No silent letters

Spell every sound you pronounce and nothing more.

### Examples:

Teroodaktl (pterodactyl),  
Nayf (knife).


## 2. Double letters are pronounced as the letter’s name

### Examples:

Waaneeng (waning),  
Aar (air),  
Aat (ate),  
Meet (meet),  
See (see),  
Bayt (bite),  
Boot (boat),  
dat (dot),
waa (way),
Dotee (duty).


## 3. Ph → F

### Examples:

Foon (phone),  
Lleefnnt (elephant).  


## 4. C → K or S (based on pronunciation)

### Examples:

Kat (cat),  
Khaar (chair),  
Sent (cent).


## 5. X → K (based on pronunciation)

### Examples:

Foks (fox),  
Ekzakt (exact).


## 6. I → E or Ay (based on pronunciation)

### Examples:

Ay (I),  
Eglo (Igloo).


## 7. Ed → T (past tense)

### Examples:

Walkt (walked),  
Jumpt (jumped).


## 8. Q → Kw

### Examples:

Kwensee (Quincy),  
Kwlefay (Qualify).


## Examples:

Original:

`The quick brown fox jumped over the lazy dog.`

English+

`Thee kwek brawn foks jumpt oovr thee laazz dag.`
